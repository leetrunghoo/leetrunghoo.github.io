
# React demo

## Used

- react
- react-router
- babel
- timeago
- sass (not using any css framework)
- sass-loader
- webpack

## How to run

__Requirement:__ installed nodejs & npm

1. Install all dependencies
	```
	npm install
	```

2. Then start webpack-server-dev
	```
	npm start
	```
